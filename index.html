Interactive Flexbox Tutorial

Two-Column Flexbox with Sticky Sidebar + Anchor Nav

0% Complete

What You'll Build

Goal: Create a two-column page with Flexbox: a 300px sticky sidebar on the left and a scrollable content column on the right. Nav links jump to section IDs.

What you'll learn:

  • Setting up a parent flex container (horizontal layout)
  • Making a fixed-width sidebar + flexible content column
  • Using anchor links to jump within the page
  • Making the sidebar sticky with CSS
  • Positioning decorative elements without blocking content

Part A ‚Äî Scaffold the HTML

A1) Start file + semantic structure

Create index.html with a semantic skeleton. Do not change class names; add your content where noted.

Your HTML should include:

  • A <header> with page title and description
  • A <main class="layout"> container
  • A <nav class="sidebar"> with 5 anchor links
  • A <div class="content"> with 3-5 sections
  • Each section needs a <div class="deco"> for decoration
  • A <footer> with basic info

HTML Structure Solution:

Part B ‚Äî Flexbox Two-Column Layout

B1) Layout container

Create styles.css and add the Flexbox parent. This sets up your two-column layout.

CSS Solution:

* { box-sizing: border-box; }
html { scroll-behavior: smooth; }
body {
  margin: 0;
  font-family: system-ui, sans-serif;
  line-height: 1.5;
}

.layout {
  display: flex;
  gap: 1.5rem;
  padding: 1rem;
}

B2) Sidebar vs. content sizing

Give the left column a fixed 300px width; let the right column flex to fill remaining space.

üí° Hint: For a fixed column, use flex: 0 0 300px;. For the flexible column, flex: 1 1 auto; works well.

CSS Solution:

.sidebar {
  width: 300px;
  flex: 0 0 300px;
  padding: 1rem;
  background: #f5f5f5;
  border-radius: 8px;
}

.content {
  flex: 1 1 auto;
  max-width: 800px;
}

Part C ‚Äî Sticky Sidebar (No JavaScript)

C1) Make the sidebar sticky

Update .sidebar to stay pinned while scrolling. This is pure CSS‚Äîno JavaScript needed!

‚ö†Ô∏è Common gotcha: Sticky works when the element's ancestor allows it (no overflow: hidden on parents). Keep .layout simple!

CSS Solution:

.sidebar {
  width: 300px;
  flex: 0 0 300px;
  padding: 1rem;
  background: #f5f5f5;
  border-radius: 8px;

  /* Sticky positioning */
  position: sticky;
  top: 1rem;
  max-height: calc(100vh - 2rem);
  overflow: auto;
}

Part D ‚Äî Anchor Links + Scroll Polish

D1) Anchor links

You already set href="#section-x" in the HTML. Make sure your sections have matching id attributes.

D2) Smooth scroll + heading alignment

Add smooth scrolling and scroll margin to prevent headings from being glued to the top edge.

CSS Solution:

html {
  scroll-behavior: smooth;
}

.panel {
  padding: 2rem;
  margin-bottom: 2rem;
  background: white;
  border: 1px solid #e0e0e0;
  border-radius: 8px;

  /* Room when scrolled-to by anchors */
  scroll-margin-top: 1rem;
  position: relative; /* needed for decor in Part E */
}

Part E ‚Äî Decorative Elements (Per Section)

E1) Add a simple, non-blocking visual

Style the .deco element so it doesn't block text or clicks. The key property is pointer-events: none.

CSS Solution:

.deco {
  position: absolute;
  top: -10%;
  left: -5%;
  width: 240px;
  height: 240px;

  pointer-events: none;   /* important! */
  opacity: 0.25;

  /* Soft gradient circle */
  border-radius: 50%;
  background: radial-gradient(
    circle at 30% 30%,
    #7dd3fc,
    transparent 60%
  );
  filter: blur(16px);
}

E2) Variation per section

Nudge the position/size per panel using :nth-of-type() to create variety.

CSS Solution:

.panel:nth-of-type(2) .deco {
  top: 50%;
  right: -5%;
  left: auto;
  background: radial-gradient(
    circle at 70% 70%,
    #fbbf24,
    transparent 60%
  );
}

.panel:nth-of-type(3) .deco {
  bottom: -10%;
  top: auto;
  background: radial-gradient(
    circle at 50% 50%,
    #a78bfa,
    transparent 60%
  );
}

Part F ‚Äî Finishing Touches (Optional)

Optional Enhancements

Polish your project with these accessibility and usability improvements:

  • Add a focus style for sidebar links (:focus-visible)
  • Style the active link based on scroll (stretch goal with IntersectionObserver)
  • Add a "Back to top" link at the end of each section

CSS Solution:

/* Focus styles for accessibility */
.nav-list a:focus-visible {
  outline: 2px solid #3b82f6;
  outline-offset: 2px;
